# CyberGames — Vercel live version

Это готовая версия для публикации на Vercel.

## Как запустить локально

```bash
npm install -g vercel
vercel dev
```

Открой:

```text
http://localhost:3000
```

## Как опубликовать

1. Создай GitHub-репозиторий.
2. Закинь туда все файлы из этой папки.
3. На Vercel нажми Add New Project.
4. Выбери репозиторий.
5. Deploy.

После деплоя сайт будет работать по публичной ссылке Vercel.

## Почему теперь будет работать

Фронтенд больше не ходит в Steam/Epic напрямую. Он обращается к своему API:

```text
/api/steam?title=Cyberpunk%202077&appId=1091500
```

А уже серверная функция Vercel ходит в Steam/Epic и возвращает:
- обложку;
- скриншоты;
- описание;
- дату релиза;
- жанры;
- цены в USD/RUB/KZT/UAH;
- отметку «Бесплатно», если Steam/Epic считает игру free-to-play.

Если игра не найдена в Steam/Epic, сайт честно показывает, что данных нет.
